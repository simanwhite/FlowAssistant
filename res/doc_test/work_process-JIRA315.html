问题描述:
For retention register, it is defined in parameter file, for example,a[31: 0]. During upf generation, the script will split the bit as a[0],a[1]...a[31].
For 2d-array register, the script could not split the bit well yet. For example, reg [3:0]a[2:0]. The script should split them as a[0][0],a[0][1],a[0][2],a[0][3], a[1][0], a[1][1], a[1][2],a[1][3],a[2][0],a[2][1],a[2][2],a[2][3].
Enhancement is needed for upf generation bit expansion script
