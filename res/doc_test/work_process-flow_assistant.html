2017-1-4

  • 一个问题:
当总是报No handler for logger 'yapsy'时, 可以加下面一段, 这样就可以看到真正的报错了:
import logging ch = logging.StreamHandler() log = logging.getLogger('yapsy') log.addHandler(ch)
说明yapsy的logging加的有点问题.
  • 一个问题:
Plugin脚本去import PyQt4写的GUI时会找不到module, 似乎说明当Plugin中import东西时, yapsy
寻找的路径还是站在当前主程序脚本的路径下去搜索的. 因此从主程序相对路径应该可以导入成功.
