vsi

VSI是SYNOPSYS开发的静态验证工具
Verdi SIgnoff platform
  • VSI-FV: Formal property checks
  • VSI-LP: Low power checks
  • VSI-CDC: Clock domain checks
这里我们主要讨论的是VSI-LP
静态验证工具还有mvsim, vclp.

VSI is a Synopsys tool.

Verdi Signoff-LP is a multi-voltage, static low power rule checker that allows e
ngineers to rapidly verify designs that use voltage control based techniques for
    power management. This document describes how Full chip and IP level VSI check
works.


VSI -LP 功能特性

  • Integrated - different signoff tools in one shell (vsi_shell) ## 集成灵活
  • Standard TCL based interface ## 标准接口
  • Separate, de-coupled Build, Check, Report steps ## 拢共三步
  • DB checkpoint/save and future restore ## DB可恢复(这也太基本了吧)
    • Enables incremental checking and reporting without re-building DB
  • Violation checks – commands to skip and configure checks ## 可配置
  • Report generation - reduce similar violations to a single representative
violation, filter/waive violations to report, by tag/family/severity/regexp
