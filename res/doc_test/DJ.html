How to Run Drop2cad
  • run all tiles' drop2cad
    • dj -J lsf -v -c -m 4 -e 'run_test "quick drop2cad__0"' -DRUN_DV=OFF
      -DDISABLE_ABV -DPUBLISH -DUNIQUIFY -DSMS -DUPF -l drop2cad_dv_only.log
      -DVIRAGE_MODELS --save_cmd_logs
  • only run some tiles' drop2cad, pls add the following to the above cmd:
    • -DPUBLISH_TILES=tile1+tile2+tile3... -DSMS_TILES=tile1+tile2+tile3...
  • #Note1: drop2cad result is under $STEM/out/linux_2.6.18_64/greenland/config/
    drop2cad_soc_greenland/pub/sim/drop2cad.out
  • #Note2: after you finish drop2cad running, you can start dj leda/cdc/lec/dc_elab
    flow.
