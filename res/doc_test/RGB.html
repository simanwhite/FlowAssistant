$摘要$:
定义:
RGB is a merge of the older SBS and ALE build tools. It can read both the .sbs
and .ale file formats.
Goals:
Provide a unified build flow for Trinity and future fusion SoCs. Unified means
all IPs use the same build flow. Accelerate the Devastator-to-Trinity
integration path, compared to the Sumo-to-Llano integration path. Support MVSIM
and VRQ flows in a uniform manner for the whole company.
